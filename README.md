# Storycoe - 绘本创

**自己做绘本，逐句跟读练口语**

## 简介

Storycoe（绘本创）是一款专为儿童设计的绘本阅读与创作应用。用户可以通过上传照片生成个性化绘本，并逐句跟读练习口语。

## 功能特性

- 📚 **绘本创作**：上传照片，自动生成绘本内容
- 🎯 **逐句跟读**：逐句朗读练习口语发音
- 🔊 **语音朗读**：支持多种语音风格朗读
- 📖 **绘本探索**：发现更多精选绘本
- 👤 **个人中心**：管理个人绘本和设置

## 技术栈

- **前端**：Flutter 3.x + Riverpod + GoRouter
- **后端**：Python FastAPI + PostgreSQL
- **AI**：Google Gemini AI

## 项目结构

```
storycoe_flutter/    # Flutter 前端应用
storycoe_backend/    # FastAPI 后端服务
```
